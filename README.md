# AVIGO Autopooling Android Application

An interactive Mobile Application Development (MAD) project built using **Android Studio (Java & XML Layouts)** designed to handle automated carpooling/autopooling routing and user coordination.

## 🚀 Key Features
* **Interactive Mapping Interface**: Features custom XML activity layouts (`activity_map.xml` & `MapActivity.java`) optimized for mapping visualization.
* **Auto Routing Calculations**: Core algorithmic components structured inside the `auto/pooling/` directories for ride grouping.
* **Responsive Material UI**: Clean user interface designed entirely with native Android XML layout components, responsive styling themes, and smooth transition animations.

## 📁 Architecture Layout
* `app/src/main/java/com/auto/pooling/` — Contains Java source components handling business logic, activity flows, and extraction rules.
* `app/src/main/res/layout/` — Holds the XML front-end structural views (such as `activity_map.xml`).
* `app/src/main/res/values/` — Contains native themes, color assets, and string definitions.

## 🛠️ Requirements & Setup
* Android Studio Jellyfish (or newer)
* SDK Platform Level 30+ (Android 11.0 or higher)
* Gradle Build System

### Installation
1. Clone this repository directly onto your machine:
   ```bash
   git clone https://github.com
   ```
2. Open the directory within **Android Studio**.
3. Let Gradle sync project files completely and press **Run** to compile the application onto your emulator or device.
